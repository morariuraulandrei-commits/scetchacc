# 🚨 ScetchACC — Schiță Accident Rutier

Aplicație web profesională pentru Biroul Rutier, pentru documentarea și schița locului accidentelor rutiere.

## 🌐 Demo Live

Dacă este publicat pe GitHub Pages: `https://morariuraulandrei-commits.github.io/scetchacc/`

## ✅ Funcționalități

### 🗺️ Hartă Live
- **Leaflet + OpenStreetMap** — GRATUIT, ZERO cheie API
- **GPS Live** — telefon, tabletă, PC
- **Căutare adresă** (Nominatim OpenStreetMap)
- Satelit (ESRI), stradal, dark, topografic

### ✏️ Schiță Autoturism, Camion, Moto, Tramvai, Pieton, Animal, Indicatoare, Marcaje

### 👥 Persoane - CNP, permis, RCA, leziuni

### 🧪 Teste - Etilotest, Drugtest, substanțe, refuz

### 📐 Măsurători

### 📄 Export PDF complet A4

### 📱 Responsive - telefon, tabletă, PC
